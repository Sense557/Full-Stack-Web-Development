MEAN Stack
MEAN is an acronym for MongoDB, Express.js, Angular, and Node.js. It's used for building dynamic web applications.
  • MongoDB: A NoSQL database for storing data.
  • Express.js: A backend framework for building APIs and handling server-side logic.
  • Angular: A frontend framework for creating interactive user interfaces.
  • Node.js: A runtime environment for executing JavaScript on the server.
MERN Stack
MERN stands for MongoDB, Express.js, React, and Node.js. It's used for building modern web applications.
  • MongoDB: The database layer for data storage.
  • Express.js: A backend framework for server-side development.
  • React: A JavaScript library for building dynamic and responsive user interfaces.
  • Node.js: The runtime environment for server-side JavaScript.
LAMP Stack
LAMP consists of Linux, Apache, MySQL, and PHP/Perl/Python. It's a traditional stack for web applications.
  • Linux: The operating system hosting the web server.
  • Apache: The web server software responsible for serving web content.
  • MySQL: A relational database for data storage.
  • PHP/Perl/Python: Server-side scripting languages for dynamic web content.
JAMstack
JAMstack stands for JavaScript, APIs, and Markup. It's a modern architecture for building fast and secure web applications.
  • JavaScript: Used for dynamic client-side interactions.
  • APIs: Access external services and data through APIs.
  • Markup: Pre-rendered markup for improved performance and security.
